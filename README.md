# 🐍 SimpleSnaker

一个使用原生 HTML5 Canvas 实现的贪吃蛇游戏，无需任何外部依赖，打开即玩。

## 🎮 玩法

- 使用 **方向键** 或 **WASD** 控制蛇的移动方向
- 吃到红色食物即可增加分数，蛇身会变长
- 随着分数提高，游戏速度会逐渐加快
- 撞到墙壁或自己的身体即游戏结束

### 快捷键

| 按键 | 功能 |
|------|------|
| `↑` `↓` `←` `→` / `W` `A` `S` `D` | 控制方向 |
| `空格` | 暂停 / 继续 |
| `R` | 重新开始 |

## ✨ 特性

- 🎨 现代暗色 UI 设计，带渐变背景和发光特效
- 📊 实时分数显示，自动保存最高纪录（LocalStorage）
- 🍎 智能食物生成，不会出现在蛇身上
- ⚡ 动态难度：每吃 5 个食物自动加速
- ⏸️ 支持暂停/继续
- 🖥️ 居中布局，适配各种屏幕

## 🚀 快速开始

直接在浏览器中打开 `index.html` 即可开始游戏。
